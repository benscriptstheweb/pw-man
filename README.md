# 🏁 Setup

## Developing
After cloning, install dependencies:
```
npm i
```
<br>

⚠️ Get required setup files from team.
<br>

Then run the application:
```
npm run dev
```
<br>

## Pushing

Create a Pull Request first. Your code will be reviewed by the team. 
<br>

### Note on commits
